Detailed Description

An object of this class maintains a connection between a ComboBox and a parameter in an AudioProcessorValueTreeState.

During the lifetime of this SliderAttachment object, it keeps the two things in sync, making it easy to connect a combo box to a parameter. When this object is deleted, the connection is broken. Make sure that your AudioProcessorValueTreeState and ComboBox aren't deleted before this object!
